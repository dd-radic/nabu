UserClassroom Table (Junction Table)
Represents the membership - which users are in which classrooms

Many-to-many relationship between users and classrooms

Tracks when users joined each classroom


User
-
ID int PK
Name varchar
Email varchar
Password varchar

FlashCard
-
ID int PK
CreaterId int FK > User.ID
ClassroomId int FK > ClassRoom.ID
Title varchar
Information varchar
Tags varchar

ClassRoom
-
ID int PK
OwnerID int FK > User.ID
Title varchar
Description varchar


UserClassroom
-
UserID int FK > User.ID
ClassroomID int FK > ClassRoom.ID
JoinedAt datetime
PK (UserID, ClassroomID)

Quiz
-
ID int PK
ClassroomId int FK > ClassRoom.ID 
CreaterId int FK > User.ID
Title varchar

Question
-
ID int PK
QuizId int FK > Quiz.ID
